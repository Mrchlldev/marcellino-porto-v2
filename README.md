# Modern Portfolio - Marcellino F.A

Portfolio modern menggunakan Next.js, TypeScript, Tailwind CSS, dan AOS Blur animation.

## Jalankan lokal

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
npm run start
```

## Setting konten

Semua teks, project, skill, education, contact, navigation, dan konfigurasi utama ada di folder:

```bash
lib/settings/*
```

Deploy langsung ke Vercel dengan import repository atau upload project ini.
